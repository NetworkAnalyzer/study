### Install Homebrew

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

### Install Python3

```
brew install python
```

### Install OpenCV

```
pip install opencv-python
pip install opencv-contrib-python
```

### Install dotenv

```
pip install python-dotenv
cp .env.example .env
```

### Install matplotlib

```
pip install matplotlib
```
