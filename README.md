# study

## description

This repo for Nojiri's thesis.

### Getting Started

1. Install Pipenv

`$ pip install pipenv`

Please add below ENV in your shell configuration file. (e.g. `~/.bashrc`, `~/.zshrc` )

`export PIPENV_VENV_IN_PROJECT=true`

2. Install dependencies

`$ pipenv install`

3. Install CLI

`$ pipenv shell`

`$ pip install -e .`

`$ study`
